Architected an OKD (OpenShift) Single Node instance for departmental testing.

- A dedicated services node configured with **HAProxy**, **BIND** (DNS), and the
  **OKD Assisted-Installer**.
- **Self-hosted GitHub Actions runners** integrated within the OKD environment
  to automate internal development workflows and CI/CD testing.
